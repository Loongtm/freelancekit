# FreelanceKit

Quotes • Invoices • Contracts — Offline-first PWA for freelancers.

## Features
- Invoices with multi-currency & locales
- Local storage, import/export JSON, print to PDF
- PWA (Service Worker + Manifest)
- Cloudflare Pages ready: `_headers`, `_redirects`

## Dev/Deploy
Just drop the folder contents into your repo root and enable Cloudflare Pages.
